# 浏览器导航技能指南

1. **区分地址栏与搜索框**：
   - 浏览器顶部的长条输入框是**地址栏**（用于输入网址）。
   - 页面中间带有放大镜图标的输入框是**搜索框**（用于搜索关键词）。
   - 当任务要求访问网址时，必须优先寻找并点击“地址栏”。

2. **输入网址防吞字符**：
   - 输入网址时，请省略 `http://` 或 `https://`，直接输入域名（如 `www.baidu.com`），以避免中文输入法冲突导致字符丢失。
